# react-native-pdf-extractor

Simple PDF ticket typeable line extractor

## Installation

```sh
npm install react-native-pdf-extractor
```

## Usage

```js
import { multiply } from "react-native-pdf-extractor";

// ...

const result = await multiply(3, 7);
```
